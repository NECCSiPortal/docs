=============================
NEC Cloud System Portal
=============================

NEC Cloud System Portal is the portal function which could excel
customization and the scalability for a cloud solution based on
OpenStack, and it has been developed based on OpenStack Horizon.

The following extension is provided as the standard function to
OpenStack Horizon.

* Business workflow:

  * Workflow engine contract management of products
  * Capacity management
  * Audit trail management

* Logging management
* Setting management
* Object storage management
* Announcement management
* Automatic construction management
